# Week 4: 물리 데이터로 학습하기 - Neural Networks GUI

| 항목 | 내용 |

---

## 2. 목적 (Purpose)

Week 4 물리 데이터 학습 실습 4개(Lab 1~4)를 **하나의 PySide6 GUI 애플리케이션**으로 통합한다.
TensorFlow/Keras 모델 학습이 백그라운드 스레드에서 실행되어 UI가 응답성을 유지하며,
학습 진행률과 결과를 실시간으로 시각화한다.

---

## 3. 사용자 스토리 (User Stories)

| ID | As a... | I want to... | So that... |
|----|---------|--------------|------------|
| US-01 | 수강생 | 1D 함수를 NN으로 근사하는 과정을 시각적으로 확인 | Universal Approximation Theorem을 직관적으로 이해 |
| US-02 | 수강생 | 포물선 운동 궤적을 NN이 어떻게 예측하는지 확인 | 물리 법칙을 데이터에서 학습하는 원리 이해 |
| US-03 | 수강생 | 과적합/과소적합 3가지 모델을 동시에 비교 | 모델 복잡도의 영향을 직관적으로 이해 |
| US-04 | 수강생 | 진자 주기를 NN이 예측하고 RK4 시뮬레이션 확인 | 비선형 물리 법칙 학습 이해 |
| US-05 | 수강생 | 학습 시작 버튼 하나로 전체 실험 실행 | 코드 수정 없이 결과 확인 가능 |
| US-06 | 수강생 | 학습 진행률을 실시간으로 확인 | 학습이 진행 중임을 인지하고 대기 |

---

## 4. 기능 요구사항 (Functional Requirements)

### FR-01: 탭 구조
- 4개 탭: [Lab 1: 1D 함수 근사] [Lab 2: 포물선 운동] [Lab 3: 과적합] [Lab 4: 진자]
- 각 탭은 독립적으로 학습 실행 가능

### FR-02: 학습 제어
- [학습 시작] 버튼 클릭 시 학습 시작
- 학습 중 버튼 비활성화
- 학습 완료 후 버튼 재활성화

### FR-03: 진행률 표시
- QProgressBar로 epoch 진행률 표시 (0~100%)
- 상태 로그 창: 현재 epoch, loss 수치 실시간 출력

### FR-04: 결과 시각화
- Matplotlib 그래프를 PySide6 위젯으로 임베드
- 학습 완료 후 결과 그래프 자동 렌더링
- 내비게이션 툴바 제공 (확대/축소/저장)

### FR-05: Lab별 시각화 내용
| Lab | 시각화 내용 |
|-----|------------|
| Lab 1 | 3가지 함수 근사 결과 + 학습 loss 곡선 |
| Lab 2 | 3가지 조건 궤적 비교 (True vs NN) |
| Lab 3 | 3모델 예측 비교 + 학습 곡선 비교 |
| Lab 4 | 주기 예측 + RK4 시뮬레이션 위상 공간 |
