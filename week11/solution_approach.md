## 문제 1·2 — H₂⁺ 전자 파동함수

### 1. 물리 모델

전자 1개 + 양성자 2개. **보른-오펜하이머 근사**: 무거운 양성자는 거리 `R` 만큼 떨어진 채 고정하고, 가벼운 전자의 슈뢰딩거 방정식만 푼다.

전자가 느끼는 포텐셜(원자 단위, 양성자를 z축 위 `±R/2`에 둠):

```
V(r) = -1/|r - R_A|  -  1/|r - R_B|
R_A = (0, 0, +R/2),  R_B = (0, 0, -R/2)
```

해밀토니안:

```
Ĥ = -½ ∇²  + V(r)
Ĥ ψ(r) = ε(R) ψ(r)
```

여기서 `ε(R)`은 전자 에너지. **총 분자 에너지**는 양성자-양성자 반발을 더한다:

```
E_total(R) = ε(R) + 1/R
```

→ `E_total(R)`을 `R`에 대해 그리면 최소점에서 **평형 결합 길이**가 나온다.

### 2. 접근법 (난이도 순으로 3단계 — 단계별로 쌓기)

#### 접근법 A — LCAO 변분법 (가장 쉬움, 먼저 구현 권장)

분자 오비탈을 두 수소 1s 오비탈의 선형결합으로 가정:

```
ψ = c_A φ_A + c_B φ_B,   φ_A = (1/√π) e^(-|r-R_A|),  φ_B = 1s on B
```

→ 2×2 **일반화 고유값 문제** `H c = ε S c` 로 환원된다 (`S`는 겹침 행렬, overlap).
필요한 적분 `S_AB`, `H_AA`, `H_AB`는 해석식이 알려져 있다(R의 함수).

```python
S = exp(-R) * (1 + R + R**2/3)
H_AA = ...   # 표준 H2+ 적분 (Coulomb)
H_AB = ...   # 표준 H2+ 적분 (resonance/exchange)
# scipy.linalg.eigh(H, S) 로 일반화 고유값
```

- **결합 오비탈**(bonding, `c_A=c_B`): 두 핵 사이 전자 밀도 ↑ → 에너지 ↓ → 결합 형성
- **반결합 오비탈**(antibonding, `c_A=-c_B`): 중앙에 노드 → 에너지 ↑
- 출력: `ε(R)`, `E_total(R)` 곡선, 평형 `R_e`, 결합/반결합 밀도 그림

**기대값(검산):** `R_e ≈ 2.0 a₀ (1.06 Å)`, `E_total(R_e) ≈ -0.60 Hartree`, 해리에너지 `D_e ≈ 2.8 eV`. (변분법이라 정확값보다 살짝 위)

#### 접근법 B — 3D 유한차분 + 희소행렬 (정공법, 가장 정확)

`01schrodinger.py`의 1D 유한차분을 **3D로 확장**한다.

- 3D 격자 `(Nx, Ny, Nz)`, 라플라시안은 7점 스텐실(각 축 2차 미분 합).
- 해밀토니안은 `Nx·Ny·Nz` × `Nx·Ny·Nz` 크기 → **밀집 행렬 불가**. `scipy.sparse`(`diags` / `kron`)로 만들고 `scipy.sparse.linalg.eigsh(H, k=4, which='SA')`로 최저 몇 개 상태만 구한다.
- 핵 근처 1/r 발산 처리: 격자점을 핵 위치에 정확히 안 놓거나(soft-core `1/√(r²+a²)`) 충분히 촘촘한 격자 사용.

```python
# 1D 2차미분 행렬을 kron으로 3D 라플라시안 조립
D2 = sparse 2nd-derivative (N,N)
I  = identity(N)
Lap = kron(kron(D2,I),I) + kron(kron(I,D2),I) + kron(kron(I,I),D2)
H = -0.5*Lap + diags(V_flat)
E, psi = eigsh(H, k=4, which='SA')
```

- 출력: 3D 파동함수의 등밀도면 또는 2D 단면(아래 C). 여러 `R`에 대해 반복 → `E_total(R)`.

#### 접근법 C — 2D 단면 시각화 (B의 부산물 / 직관용)

양성자 두 개를 포함하는 평면(예: xz 평면)에서 2D 유한차분으로 풀거나, B의 3D 해를 평면으로 잘라 `|ψ|²` 히트맵을 그린다. 결합 오비탈에서 두 핵 사이가 밝게, 반결합에서 중앙에 어두운 노드선이 보이는지 확인.

## 문제 3 — 중성 He (전자 2개): 풀 수 있다, 단 몬테카를로로

### 1. 왜 He⁺보다 훨씬 어려운가

He⁺(전자 1개)는 그냥 Z=2 수소형 문제라 해석해가 있다(`E = -2.0 Hartree`).
하지만 **중성 He는 전자가 2개**라서 해밀토니안에 **전자-전자 반발항** `1/r₁₂`가 들어간다:

```
Ĥ = -½∇₁² - ½∇₂²  - 2/r₁ - 2/r₂  + 1/r₁₂
```

- 파동함수가 `ψ(r₁, r₂)` → **6차원**. 격자 유한차분이면 `N⁶` 크기라 사실상 불가능.
- `1/r₁₂` 때문에 **변수분리가 안 됨**(비분리 다체 문제). 정확한 해석해 없음.

→ 그래서 **변분법 / Hartree-Fock / 몬테카를로**가 필요하다.

### 2. 접근법 (단계별로 쌓기)

#### 접근법 1 — 단순 변분법 (워밍업, 해석적)

시험 파동함수 = 유효전하 `Z_eff` 의 1s 두 개 곱:

```
ψ(r₁,r₂) = e^(-Z_eff·r₁) · e^(-Z_eff·r₂)
```

에너지 기댓값은 해석식 `E(Z_eff) = Z_eff² - (27/8)Z_eff` → 최소화하면

```
Z_eff = 27/16 = 1.6875,   E = -(27/16)² = -2.848 Hartree
```

전자가 서로를 가려(screening) 유효 핵전하가 2보다 작아진다는 직관을 준다.
- `Z=2`(가림 무시) → `E = -2.75 Hartree`
- `Z_eff=1.6875`(가림) → `E = -2.848 Hartree` (정확값 −2.9037에 접근)
- **구현**: `scipy.optimize.minimize_scalar`로 `E(Z_eff)` 최소화. 빠르고 e-e 상관의 절반(가림)을 보여줌.

#### 접근법 2 — 변분 몬테카를로 (VMC) ★ 핵심, "몬테카를로 뭐시기"

상관(correlation)까지 넣으려면 **Jastrow 인자**가 붙은 시험함수를 쓰고, 6차원 적분을 **메트로폴리스 샘플링**으로 계산한다.

시험 파동함수(He 표준형):

```
ψ(r₁,r₂) = e^(-Z r₁) · e^(-Z r₂) · exp( r₁₂ / (2(1 + b·r₁₂)) )
```

- `Z`, `b` : 변분 파라미터. Jastrow 항 `exp(r₁₂/...)`이 두 전자가 가까울 때 확률을 높여 **상관**을 표현하고 cusp 조건을 만족.

**VMC 알고리즘 (메트로폴리스):**
1. 두 전자 위치 `(r₁, r₂)` (총 6차원)를 무작위 초기화.
2. 한 스텝마다 위치를 조금 흔들어 제안 → 확률 `min(1, |ψ_new|²/|ψ_old|²)`로 수락(Metropolis). 이렇게 하면 표본들이 `|ψ|²` 분포를 따른다.
3. 각 표본에서 **국소 에너지** `E_L = (Ĥψ)/ψ` 계산. He는 `E_L`의 해석식이 알려져 있어 라플라시안을 수치미분 없이 바로 쓸 수 있다(Thijssen 교재 식).
4. `⟨E⟩ = 평균(E_L)`, 통계오차는 표본 분산/√N.
5. 바깥 루프에서 `Z, b`를 바꿔 `⟨E⟩`를 최소화(수동 스캔 또는 간단한 최적화).

**기대값(검산):**
- 단순 변분 `-2.848` → VMC(Jastrow) `≈ -2.878 Hartree` → 정확값 `-2.9037 Hartree = -79.0 eV`.
- 첫 이온화 에너지 = `E(He⁺) - E(He) = -2.0 - (-2.90) = 0.90 Hartree ≈ 24.6 eV` (실험값 24.59 eV)로 검산.

#### 접근법 3 — (선택) Hartree-Fock / DMC

- **Hartree-Fock(자기무리장, SCF)**: 각 전자가 "평균장"을 느끼는 1전자 방정식을 자기일관적으로 반복해서 푼다. He는 한계값 `-2.862 Hartree`. 상관은 못 잡지만 평균장 다체법의 표준.
- **Diffusion Monte Carlo(DMC)**: 허수시간 진화로 바닥상태를 사실상 정확히 투영. VMC의 확장. 시간 되면 도전.
