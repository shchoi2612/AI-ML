# PRD: EconSim — 국가 경제 시뮬레이션 게임

## 한 줄 요약

20개월 임기의 국가 경제 수장이 되어 정책을 선택하면, 시장이 실시간으로 반응하고 LLM이 뉴스로 해설하며, 임기말에 EMH 성적표를 받는 웹 게임.

## 목표

| 항목 | 내용 |
|------|------|
| 타입 | 5주 코스 프로젝트 (2인 팀) |
| 평가 기준 | 수익성·타당성 > novelty |
| 산출물 | GitHub repo, 데모 영상, 10슬라이드 PPT, 아키텍처 포스터 |
| Hero 순간 | 정책 선택 → Lightweight Charts ETF 차트 실시간 반응 (3초 이내) |

## 차별화

Democracy 4와 시뮬레이션 대상 자체가 다르다. Democracy 4는 "유권자가 나를 재선시킬 것인가"(정치적 지지), 이 게임은 "시장이 내 정책을 어떻게 가격에 반영하는가"(금융시장 반응). 정치 시뮬레이터엔 금융시장 레이어도 학술 검증 레이어도 없다.

## 게임 플로우

```
게임 시작 (임기 시작)
    │
    ▼
┌─────────────────────────────────────────┐
│  매 월(턴) 반복:                         │
│                                          │
│  1. 현재 상태 표시 (게이지 4개 + ETF 5종) │
│  2. 위기 이벤트 발생                      │
│  3. 정책 선택지 3개 (힌트 표시)           │
│  4. 선택 → 게이지/ETF 즉시 계산 (<1초)    │
│  5. 차트 실시간 업데이트 (hero 순간)       │
│  6. LLM 뉴스 해설 스트리밍                │
│  7. VN 말풍선 페르소나 반응               │
│  8. 파면 조건 체크                        │
└─────────────────────────────────────────┘
    │
    ▼
임기 완주 (20개월) 또는 파면
    │
    ▼
임기말 성적표 (안정도 점수 + EMH 분석)
```

## 게이지 (HP 대체)

| 게이지 | 초기값 | 파면 조건 | 위험 방향 |
|--------|--------|-----------|-----------|
| 부채 | 50 | ≥ 100 (국가 부도) | 높을수록 위험 |
| 인플레이션 | 40 | ≥ 100 (초인플레) | 높을수록 위험 |
| 민심 | 60 | ≤ 0 (혁명) | 낮을수록 위험 |
| 국제긴장도 | 30 | ≥ 100 (전쟁) | 높을수록 위험 |

## ETF 5종 (보조 지표)

반도체, 에너지, 금융, 방산, 소비재. 정책 결과에 따라 민감도 테이블 기반으로 가격 변동. 게이지 변화량 x 가중치 + 노이즈.

## 입력 모델

스타일드 선택지 피커. 이벤트마다 3개 선택지 버튼. 채팅 UI 스타일로 렌더하되 실제는 choice_index 전송. 자유 텍스트 파싱은 MVP 범위 밖.

## LLM 나레이션

Groq llama-3.3-70b-versatile. 계산된 숫자를 받아 한국어 경제 뉴스 2~3문장 생성. 엔진이 숫자 계산, LLM은 해설만. API 실패 시 템플릿 폴백.

## VN 연출

AI 생성 포트레이트 3~5장 (재무장관, 중앙은행 총재, 야당 대표, 외교관, 기자). 말풍선 + 타이핑 연출. 풀 VN 아님. 엔진과 분리된 뷰 컴포넌트.

## EMH 검증 (학술 레이어)

Pearson 상관계수: 정책 카테고리별 → 해당 섹터 ETF 변동률 상관 분석. 임기 완주 시 1페이지 성적표로 표시. "당신의 정책은 시장에 X% 예측 가능한 패턴을 만들었다."

## 승리/패배

- 승리: 20개월 생존 → 안정도 점수 (최대 400점) + EMH 성적표
- 패배: 게이지 한계 도달 → 파면 사유 표시

## 범위 밖 (NOT in scope)

- 자유 텍스트 정책 입력 (LLM 파싱)
- 다국가 동시 플레이
- 유료 규칙/과금
- 딥 EMH 대시보드 (Granger causality 등)
- 풀 비주얼 노벨 (분기, 호감도)
- 모바일 대응
